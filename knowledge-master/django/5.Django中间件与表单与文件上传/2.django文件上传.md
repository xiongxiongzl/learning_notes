### 前言

在很多的应用场景中都需要上传图片，图片可以作为头像，文章的展示图片，或者web的banner图片等等，这时候就需要了解django中怎么去设置图片上传，并且实现该上传图片的功能。

#### 1. 安装PIL库

```
pip install Pillow
```
